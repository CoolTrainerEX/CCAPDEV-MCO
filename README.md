# ArchersLab

The app was made with [Bun](https://bun.com/), but should be compatible with [Node.js](https://nodejs.org/en).

## Setup

First run the following setup commands to generate types:

```bash
bunx --bun orval
bunx --bun prisma generate
```

or

```bash
npx orval
npx prisma generate
```

## Run

To run the app, run:

```bash
bun run build
bun start
```

or

```bash
npm run build
npm start
```
